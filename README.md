# 🖋️ จดแล้วรวย (Durian Scale PWA)

แอปบันทึกน้ำหนักและราคาขายทุเรียนแบบ **PWA** (Progressive Web App) ติดตั้งลงมือถือและใช้งานออฟไลน์ได้ เหมาะสำหรับจดตอนชั่งทุเรียนหน้าสวน

## ✨ ฟีเจอร์

- **เลือกเกรดก่อน → กรอกน้ำหนักช่องใหญ่** ใช้ง่ายขณะชั่งจริง
- แบ่งเกรด **AB / C / ตกไซส์ / อื่นๆ** และ **เพิ่มเกรดเองได้**
- **ตารางแยกตามเกรด** สรุปย่อยอัตโนมัติทุก 5 แถว (ชุดละ 5)
- **สรุปราคา** กรอกราคาต่อกิโล คำนวณเป็นเงินให้อัตโนมัติ
- **หักค่าใช้จ่าย** (ค่าแรง / ค่ารถ / ค่าตัด ฯลฯ) ได้ไม่จำกัด
- **คงเหลือสุทธิ** = ยอดขาย − ค่าใช้จ่าย
- บันทึก **สวนที่ตัด + วันที่ + เวลา** ทุกครั้ง
- **ประวัติ** พร้อมสรุปยอดรวม **รายวัน / รายสวน**
- **ส่งออก CSV** เปิดใน Excel ได้
- **บันทึกร่างอัตโนมัติ** กันข้อมูลหายระหว่างกรอก
- ใช้งาน **ออฟไลน์** ได้เต็มรูปแบบ (Service Worker)

## 🚀 การใช้งาน

เป็นเว็บ static ล้วน ไม่ต้อง build เปิดผ่าน HTTP server ใดก็ได้:

```bash
# ตัวอย่างด้วย Python
python -m http.server 8080
# แล้วเปิด http://localhost:8080
```

> ต้องเปิดผ่าน `http(s)://` หรือ `localhost` (ไม่ใช่ `file://`) เพื่อให้ Service Worker ทำงาน

### ติดตั้งลงมือถือ
เปิดเว็บในมือถือ → เมนูเบราว์เซอร์ → **เพิ่มลงในหน้าจอโฮม / Install**

## 📂 โครงสร้าง

```
durian-scale/
├── index.html       # หน้าแอป
├── styles.css       # ดีไซน์ (มือถือเป็นหลัก)
├── app.js           # ตรรกะทั้งหมด
├── manifest.json    # ข้อมูล PWA
├── sw.js            # Service Worker (ออฟไลน์)
└── icons/           # ไอคอนแอป
```

## 🛠️ เทคโนโลยี

HTML + CSS + Vanilla JavaScript ล้วน ไม่มี dependency เก็บข้อมูลด้วย `localStorage` บนเครื่องผู้ใช้
